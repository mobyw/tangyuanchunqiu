难忘的一九五七年

| | | | |

【老顽童按】编辑完孺子牛的唐院整风和反右运动史记后，仍犹感不足，似乎意犹未尽。余以为，历史事件只有结合相关的历史人物来描写，才生动、有血有肉。而且，要想还原一段历史，须得全方位、多视角、有不同阶层人的综合描述，才能言及接近历史真实。而原文缺少从投入运动的唐院师生的视角的观察、理解、感触和遭遇结果。再看运动领导者的视角又缺少批判者的高论以及当年“胜利结束”这场运动时对我校的具体结论性意见。

于是乎，老顽童便分别向原作者和曾经历过此运动的学长们约稿。感谢学姐茴香斋主慨然应允，确令我感佩不已，学姐文笔清新，文思严谨，文采斐然。更重要的其为人求实求是，重考证，轻臆断，此乃纪实行文之大乘也。

在此，老顽童谨代表唐院春秋编辑部再次感谢茴香斋主校友的倾力支持！

难忘的一九五七年
作者／茴香斋主

一，前奏

1956年春天，各国共产党的老大哥苏共展开了对斯大林的批判，“解冻”成为潮流。我国党中央宣布在文学艺术和自然科学领域采取允许“百花齐放，百家争鸣”的方针。也就是所说的“双百方针”。中央宣传部长陆定一代表中央宣布每个人都有“独立思考的自由，有辩论的自由，有创作和批判的自由，有发表自己的意见，坚持自己的意见和保留自己的意见的自由。”不少人耳目为之一新，认为跨进了一个新的时代。同时中央又发出了“向科学进军”的号召，要求有关部门和单位编制“十二年科学规划”，个人也要提出自己的规划，学校教师，特别是年青教师，都满怀信心，劲头十足，提出自己的规划，除了完成教学任务之外，每年要写出多少篇科研论文，几年内要完成那些研究专题和项目。几年内要使自己达到副博士水平，又要几年内达到博士水平等等。

学校文体活动也活跃起来，一扫头年肃反运动留下的肃杀之气。学校相继举行了冰上运动会，田径运动会，进行了歌咏比赛，足球比赛等。我们桥班班主任还拿出他的一些唱片，课余组织我们桥班的同学欣赏，唱片有冼星海的《黄河大合唱》，柴柯夫斯基的《天鹅湖》，约翰斯特劳斯的《圓舞曲集》，古诺，舒伯特，海顿的《小夜曲》等。隧道教研组的麦倜曾讲师，为我们桥班和隧班的同学，课余主持了西方古典音乐欣赏讲座，他是西方古典音乐的爱好者，解放前夕，在北平一大学的土木系毕业时，正置北平解放前夕，找不到工作，就到中央音乐学院去学习了一年半，到50年初才到唐院任教，因他受过正规的音乐训练，音乐水平较一般学工的人要高，他收集有不少的西方古典音乐的唱片，那时这种唱片发行量少，物以稀为贵，所以价格也较贵。他给我们介绍了一些著名音乐家的作品，边放唱片并加以解说，如贝多芬的《月光奏鸣曲》，《热情奏鸣曲〉，萧邦的《前奏曲》，《即兴幻想曲》，李斯特的《匈牙利狂想曲》，柴柯夫斯基的《1812年序曲》，《悲怆奏鸣曲》，贝多芬的《命运交响曲》，《田园交响曲》等等，分段讲解了序曲，主题，乐章等等，他讲得津津有味，我们听得云山雾罩，但不管怎么说，对我们还是起到了一定的启蒙作用。另外，校外的电影院放映的影片，也放映了一些除苏联电影外的外国电影，如印度电影《流浪者》，墨西哥电影《生的权利》，阿根廷电影《中锋在黎明前死去》等就大受观众欢迎。

年底前，北京中央音乐学院教师带领学生，到我们学校作毕业实习的音乐会，演出是在明诚堂举行，盛况空前。看来，麦讲师给我们的讲座，派上了用场。该校声乐教授，声乐系主任，著名女高音歌唱家喻宜萱，演唱的一曲《康定情歌》，引起全场轰动。学生们的演出，声乐方面的节目有，男声，女声独唱，男声, 女声，混声四重唱，以及男声，女声，混声三部合唱，器乐方面的节目有，小提琴奏独奏，中提琴独奏，长笛独奏，钢琴独奏，以及弦乐四重奏，钢琴三重奏等，都赢得热烈的掌声，且服装华丽，台风优美，这是我第一次观赏西方音乐的演唱会，留下了难忘的印象。

新年前夕，学校里举办了新年化装舞会，同学们穿上了五彩缤纷的各少数民族的服装，随着欢快的乐曲翩翩起舞。一些青年教师也来参加，不过他们没有化装，我班来自西贡的华侨女同学装扮成，头戴尖顶绒饰小帽，腰着横纹围裙的藏族姑娘，我装扮成梳着很多小辫，戴着小花帽的，新疆维吾尔族的姑娘。在热情洋溢的欢乐气氛中，迎接新的一年的到来。

二，主题

57年春季开学，是我们这年级的最后一学期，原定从二月中，学期开始就进行毕业设计至六月中完成，留下最后一个月，进行毕业设计答辩，毕业鉴定，以及毕业分配，七月中毕业离校。毕业设计的内容是，在给出的各自不同的原始资料下，各人独立进行一座中型桥梁的设计，完成从初步设计到技术设计至施工设计，包括写出设计说明书，绘制出设计图纸，编制出概预算，订出总的工程进度表等。在上学期上桥梁这门课时，虽然也作过桥梁的课程设计，但那只是作部分的，较小型的桥梁设计，作业量也没这么多，也不需要进行答辩。而毕业设计内容多，工作量大，时间紧，同学们大家都日以继夜地，投入到毕业设计的工作中来。

正当我们开始投入作毕业设计的时候，政治形势已开始有了变化，党中央提出，要求全民帮助党“整风”，要求大“鸣”大“放”。而不仅是文化、艺术、科学领域学术问题的“百花齐放，百家争鸣”的“鸣”、“放”了。而是政治上的“不平则鸣”的“鸣”，“有气要放”的“放”了。

虽然党中央鼓励人们“大胆鸣，大胆放”，“知无不言，言无不尽。言者无罪，闻者足戒。”，但开始反应并不热烈，相当多的人仍然怀有疑惧，党是否会真正实行“言者无罪”，是否会“秋后算帐”等。中央民族学院社会学家费孝通，三月中首先在《人民日报》上发表了题为《知识份子的早春天气》，文中道出了大多数知识份子的心理状况：“心里热，嘴却很紧……怕是个圈套，搜集些思想情况，等又来个运动时可以好好整一整”。四月里，北京大学历史学家翦伯赞发表了短文，谈到多数知识份子对中央号召“鸣放”的态度：“他们还在揣测，这是真放还是假放。如果是真放，放多少，放了以后还收不收，放是手段还是目的。”上海复旦大学教授王造时发表对《光明日报.》记者的谈话说到：“希望中央发表一个比较具体的声明，保证除现行反革命份子外，一切思想问题都不在追究之列。”。有人在谈论人们为何不愿鸣放时，引用了唐诗“含情欲说宫中事，鹦鹉前头不敢言”，道出了多数知识份子的心声。

要求鸣放的前一阶段，我们一心紧张地作毕业设计，对于鸣放问题不大关注，但从北京大学第一张鸣放大字报于五月十九日贴出起，鸣放处于高潮阶段，《人民日报》上接连登载有各民主党派，文艺界，学术界在鸣放座谈会上的言论，才引起了注意，其中比较突出的有，交通部长，全国政协副主席，民盟副主席章伯钧提出：“……工业方面有许多设计院，可是政治上的许多设施，就没有一个设计院，我看政协，人大，民主党派，人民团体，应该是政治上的设计院……，一些政治上的基本建设，要事先交他们讨论。”后来称之为“政治设计院”。国务院森林工业部部长，民盟副主席罗隆基主张：全国人大和政协成立一个委员会，而且建议全国上下都设立这种委员会，形成一个系统，为历次运动中受了冤曲的人平反。后来称之为“平反委员会”。中华人民共和国国防委员会副主席，民革副主席龙云的“反苏”言论：“……中国抗战胜利，日本投降。苏联出兵东北，将东北价值二十亿美元的机器和物资，搬去了苏联，至今未归还，怎么算是友好和兄弟国家？”，“……朝鲜战争是苏联指使北朝鲜发动的，反要中国去抗美援朝，而所用的武器又是用中国的物资，尤其是粮食，向苏联换取的，一切战费皆由中国负担，这样是公平合理吗？”。《光明日报》总编辑储安平在统战部召开的一次座谈会的发言，他说：“我认为党领导国家并不等于这个国家即为党所有。但是，在全国范围内，不论大小单位，甚至一个科，一个组都要安排一个党员做头儿……。党这样做，是不是‘普天之下，莫非王土’那样的思想，从而形成了现在这样一个一家天下的清一色局面，……”后来将他的言论概括为“党天下”。剧作家吴祖光`在《戏剧报》上发表文章题为“党‘趁早别领导艺术工作’”，文中说：“行政领导看戏，稍有不悦，艺人就回去改”“一切都是领导决定的，甚至每一个艺术处理，剧本修改也都是按领导意图作出来的。”“今天无数的艺术团体的领导，偏偏就不估计人民群众的需要，认为群众浑噩无知。对群众喜爱的东西，用无数清规戒律斩尽杀绝。”“如果党一定要这样领导，过去是谁领导屈原的？谁领导李白，杜甫，关汉卿，曹雪芹，鲁迅的？又谁领导莎士比亚，托尔斯泰，贝多芬，莫里哀的？”。民革女中央常委谭惕吾在鸣放会上发言，批评共产党带头违反宪法，指出城市对房产的‘改造’是违法的，她说“共产党必须遵守宪法，不应在法律、法令之外，再发内部指示，用指示代替法律、法令是不可以的……”。中国人民大学的女学生林希翎在北京大学的辩论会上发表自己的观点：“……我认为我们现在的社会主义不是真正的社会主义，真正的社会主义应该是很民主的，但我们这里是不民主的，我们管这个社会叫做在封建基础上产生的社会主义，我们要为一个真正的社会主义而斗争……”。在作毕业设计的间歇，从班上订的《人民日报》上看到这样一些言论，有些觉得新颖，有些还觉得不无道理。因当时自己并没有什么成熟的政治见解。

到学校食堂吃饭时，陆续看了在食堂贴出的大字报，但多半是就事论事，对具体事例表示不满。其中集中意见最大的，是学校的迁校问题，新校址长期没定下来，影响了学校的发展，于是就有人提出成立一个建校委员会，来促进建校问题的解决。于是就推选出由教授和学生代表，组成的建校促进委员会，并推举出十多人的代表团到北京，直接向铁道部领导提交书面意见，这代表团去京后数日回校，将代表团到北京后向铁道部请愿的经过情况，用大字报的形式公布出来，才知铁道部开始时不予理采，后来不得已由铁道部副部长刘建章出来接见，但刘副部长根本不听取代表团成员的合理要求，反而一味地打官腔，也不想解决问题，就将代表团打发回唐山。这一大字报贴出后，一时间群情激愤，食堂里就贴出很多针对刘建章的大字报。在我们作毕业设计制图室的墙璧上，也有同学贴出一些小字报进行鸣放，也有发表不同观点的。但没有给我留下深刻的印象。大多数同学，主要精力投入毕业设计，写小字报的很少。大都是抽空看班上订的《人民日报》，来关注运动的发展，但也不去评论。

到了六月初，鸣放言论趋于激烈的同时，《人民日报》已悄悄开始变调，在六月六日发表文章说“帮助共产党整风，是为了巩固党的领导”“……有些人在反对教条主义的旗帜下，反对马列主义……”。六月七日的文章说“……共产党的领导不容动摇，社会主义方向不容模糊……”。六月八日反右斗争正式揭幕，《人民日报》发表题为《这是为什么》的社论，指出：“少数的右派份子正在向共产党和工人阶级的领导权挑战”。“他们企图乘此时机把共产党和工人阶级打翻，把伟大的社会主义事业打翻，拉着历史向后倒退，退到资产阶级专政，……把中国人民重新放在帝国主义及其走狗的反动统治之下。”。六月十日，《人民日报》刊登社论，《工人阶级说话了》，于是全国各地工人动辄举行数千，上万人的集会，“愤怒声讨反党反社会主义的右派份子”。农民也说话了：“想取消党的领导，就是要拆散农民的集体幸福生活。”。工人、农民们“一千个不答应，一万个不应”。《人民日报》从六月中旬连篇累牍刊登了辱骂右派的文章。另外，《人民日报》开辟题为“什么话”的专栏，专登“右派言论”供批判用。

我们的毕业设计于6月中完成后，就交给系答辩委员会审阅。我们都积极准备7月初进行的毕业设计答辩，系答辩委员会是由系主任及各专业课教师组成。7月中我们毕业班完成了毕业设计答辩，就接到学校通知，全校毕业年级的学生，推迟两个月毕业，参加“反右”运动后，再进行毕业鉴定和毕业分配。这时 “右派份子”们的认罪书，已开始纷纷登载在报刊上，如费孝通、储安平、黄琪翔等人的检讨书被冠以“向人民伏罪”，“向人民投降”，“请求人民宽恕”的标题登在《人民日报》上。而对那些拒不认罪，甚至以死抗争的“右派份子”当然是不会登在报上的。

我们学校的“反右斗争”和全国一样，大张旗鼓地开展起来了，通过一个多月的大揭发，大批判，大斗争，一个个“右派份子”，给“挖”出来了。主要有：学院白铁岩副院长，机械系系主任孙竹生教授，每个系都有多位教师被罗织其中，而“建校委员会”的十几位成员，无论是教授还是学生，几无一幸免。以曾给我们班授过课的教师来说就有：讲授《高等数学》的郭可詹教授，讲授《材料力学》的牛清江付讲师，，辅导《电工原理》的徐慰农助教，讲授《中国革命史》的胡思杜讲师。《人民日报》八月二十一日发文报道，唐山铁道学院机械系主任孙竹生，及胡思杜两人“使用卑鄙手段妄图夺取学校领导权”后，胡思杜讲师自杀身亡，学校领导说他是畏罪自杀。胡思杜是去了台湾的文化名人胡适的儿子，胡适的子女中，只有他一人选择留在大陆，并于解放初，就公开宣布与“反动”的父亲划清界限。学生中，几乎每个系，年级，班级都有人落网。总数超过百分之十。

以我们班来说，总共三十来人，就定了四名“右派份子”，像行将离巢的初飞燕，还没有飞就被折断了翅膀。象一场噩梦一样，一夜之间，昔日的老师，同学，成为了“十恶不赦”的阶级敌人。对这场迅雷不及掩耳的运动，我不知如何来形容我当时的震惊与困惑。

三，终曲

学校的反右运动结束后，我们年级就开始进行毕业鉴定，毕业鉴定先是作自我总结，主要是结合反右运动中自己的立场，观点，表现，以及检查思想上对那些右派言论有同情，有共鸣等进自我行批判，人人过关，人人自危，然后是群众评议，批判，班委会作出评语，最后是领导给予审查意见。虽然没有前一阶段反右斗争那样声势大，但气氛紧张，不知道什么时候厄运会降临到自己头上。我虽然对报纸上登载的反右情况相当关注，但未曾与人谈论，但看了报，思想上不能没有想法，一点不“检查” 也是不行的，我就“检查”了对一些右派言论“认识不清”，“认识模糊”，“革命警惕性不高”，另外就是“检查”了，认为铁道部副部长刘建章太官僚，不关心我们学校的“迁校”与“建校”问题，这个部长就不要当好了。而看不出右派借此大作文章等。总算过关，大大松了一口气。

毕业鉴定完后，接着就是进行毕业分配，因是在反右运动以后，这一年的分配方案不同于往年，是强调下基层，学校的分配方案中，没有学院和设计院，也没有研究单位，只有工程局和管理局,，有一研究生名额已内定。分配名单一公布，同学们就各自上路了，没有聚会，没有告别，没有祝福，带着心灵的伤痛，就这样黯然而去。原先设想，对于完成学业，进入社会，展开人生新篇章，当好好庆祝一番，未曾想却是“无言空相觑”，但此时“无声胜有声”。四年多前的豪情，半年多前的憧憬不复存在。但想想四年来，母校的培育，师长的教诲，同窗的砥厉，历历在目。怀想母校数经战乱之灾，备尝流离之苦，杏坛断续，仍弦歌未绝，唐山子弟，遍及华林。作为一个唐山子弟，即使荆棘载途，想亦能开辟出新天地。
